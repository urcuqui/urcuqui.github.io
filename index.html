SafeCandy is a client-server architecture system to analysis and to detection of Android malware through a modification of the ASEF. The system implements both static and dynamic analysis of AVDs’ environments architecture. Additionally, the solution implements a cloud architecture that connects various malware databases, for which decision-making. SafeCandy implements various advantages found in the study of the state of the art; in order to measure the performance, we demonstrated its capacities through an experiment that includes different malwares and beningn applications (they are from malicious databases and Google Play respectively).

This project is a forward of safe candy. During this time we researched and developed a framework to detect malicious software in Android, integrated by static analysis through the evaluation of permissions of benign and malware



It is a project that search to develop a security system to analysis and to prevent the website defacement in public entities sites. Indeed, during these times we explored the state of the art, we worked with some public entities and we proved our system through some experiments with real infraestructures

This project has two focuses, firstly to apply an analysis of anomalies and to detection of these in network traffic, and secondly a reseach of how to identify malicious websites through the analysis in web traffic
